Create the class ProductStock with immutable properties: val name: String and val code: String. <br/>2. Define the property stockQuantity: Int with an initial value of 0 and the private modifier. <br/>3. Create a public method recordStockIn(quantity: Int) to increase the stock. It must only accept positive values. <br/>4. Create a public method recordStockOut(quantity: Int) to decrease the stock: <br/>a. It must verify that the quantity to be removed is not greater than the current stockQuantity. <br/>b. It returns a Boolean (true for success or false for failure). <br/>5. Create a public method getQuantity() to read the current stock value. |
